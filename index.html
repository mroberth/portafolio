¿Quién es Roberth Matos?

Estoy a un solo semestre de graduarme como Ingeniero en Informática. Mi vocación nació de la fascinación por la tecnología y las computadoras desde muy pequeño, alimentada también por mi lado gamer.

Mi alias Trocknickero derivó en la marca digital trocknick.dev, el sello técnico donde combino lógica, diseño estético e ingeniería.

Cómo Trabajo

No me conformo con soluciones que solo "funcionan en la superficie". Soy perfeccionista y disfruto desglosar los problemas paso a paso: desde el diseño de la base de datos hasta la seguridad de la API y la experiencia de usuario.

Para mí, utilizar herramientas avanzadas o IA es una ayuda para agilizar, pero el verdadero valor de un ingeniero radica en comprender a fondo cada línea de código y saber explicar detalladamente el porqué de cada decisión.

CEV Informática — Sistema de Gestión Académica

Plataforma web integral desarrollada para el PNF en Informática. Permite administrar estudiantes, docentes, notas, asignaciones y horarios con soporte para 3 roles de usuario.

  • Seguridad: Autenticación JWT, protección CSRF y Rate Limiting (Token Bucket).
  • Tiempo real: Notificaciones mediante Server-Sent Events (SSE).
  • Reportes: Panel interactivo con gráficos de rendimiento con Chart.js.

DIRPOLES_IA — Microservicio de Análisis Estadístico

Motor analítico en Python con FastAPI diseñado para conectarse de forma independiente con el sistema institucional DIRPOLES_4 (PHP).

  • Arquitectura desacoplada: Comunicación HTTP/JSON con autenticación por API Key.
  • Procesamiento de datos: Análisis mensual, tendencias, morbilidad y resúmenes ejecutivos.
  • Validación estricta: Modelos con Pydantic y documentación OpenAPI/Swagger automática.
