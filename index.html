Ecosistema DIRPOLES (DIRPOLES_4 + APP + IA)

Ecosistema institucional completo compuesto por un monolito web en PHP, una aplicación móvil conectada y un microservicio analítico en Python (FastAPI) integrado con Google AI Studio (Gemini API) para la generación de reportes estadísticos e informes inteligentes.

  • Monolito DIRPOLES_4: Sistema central para la administración de datos, servicios e inventarios.
  • App Móvil DIRPOLES_APP: Interfaz adaptada a dispositivos móviles para interacción de usuarios.
  • Microservicio DIRPOLES_IA: Procesa reportes mediante consultas inteligentes a Google AI Studio para emitir hallazgos, métricas y recomendaciones automáticas.

CEV Informática — Sistema de Gestión Académica

Plataforma web integral desarrollada para el PNF en Informática. Permite administrar estudiantes, docentes, notas, evaluaciones y horarios con soporte para 3 roles de acceso.

  • Seguridad: Autenticación JWT, protección CSRF (Double-Submit Cookie) y Rate Limiting.
  • Tiempo real: Notificaciones continuas mediante Server-Sent Events (SSE).
  • Estadísticas: Paneles con gráficos interactivos para evaluar el rendimiento académico.
